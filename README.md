## How to run locally
1. Clone:
git clone https://github.com/krishnaa991/PRODIGY_WD_04.git
2. `cd PRODIGY_WD_04`
3. Open `index.html` in your browser.

## Deployment (GitHub Pages)
1. Push repo to GitHub.
2. Settings → Pages → Select `main` branch and `/ (root)` → Save.
3. Visit `https://krishnaa991.github.io/PRODIGY_WD_04/` after deployment.
